## 프로젝트 소개

<p align="justify">
프로젝트 개요/동기
<br>
계산기라는 컨셉 아래에 다양한 계산기 기능을 제공
<br>
디데이계산기와 퍼센트계산기 기능 구현
</p>

<br>

## STACK

Vue
<br>
TailwindCss
<br>
Vue-router

<br>

## 구현 기능

### 기능 1
메인 페이지에서 퍼센트와 디데이계산기를 선택하여 이동할 수 있도록 라우터 설정
### 기능 2
각각 계산기의 기능 구현
### 기능 3
디데이 계산기에서 디데이날짜를 input type text로 사용자에게 전달 받았을 때 잘못된 형식의 날짜를 입력했을 때의 예외처리를 고려하여 input type date로 설정
